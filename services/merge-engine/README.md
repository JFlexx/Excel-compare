# services/merge-engine

Espacio reservado para el servicio responsable del procesamiento principal de comparación y resolución.

## Responsabilidades esperadas

- ingesta y normalización de workbooks;
- generación de diffs estructurales y de contenido;
- identificación de conflict y cambios auto-resolubles;
- aplicación de decision para construir el result workbook;
- contratos de entrada y salida para la merge session.
