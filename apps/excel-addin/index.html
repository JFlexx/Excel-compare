    Uso estándar

    • Hasta 25 MB por archivo.
    • Hasta 25 hojas por libro.
    • Hasta 200.000 celdas relevantes por comparación.

    • Hasta 50 MB por archivo con degradación controlada.

    • Si se exceden los límites, la operación se bloquea con mensaje y referencia para soporte.

    Alcance inicial

    • Prioridad: Excel Desktop para Windows con Microsoft 365.
    • Alternativa controlada: Excel 2021 LTSC aprobado por TI.
    • Fuera de alcance inicial: Excel para Mac sin validación expresa.
